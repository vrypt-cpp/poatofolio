# vrypt — Portfolio

React + Vite portfolio, dapat di-build sebagai static site.

## Setup

```bash
npm install
npm run dev      # development server
npm run build    # build ke folder dist/ (static, siap deploy)
npm run preview  # preview hasil build
```

## Edit konten

Semua data profil, stack, dan project ada di `src/data.js` — cukup ubah file ini untuk memperbarui konten portofolio.

## Deploy

Folder `dist/` hasil build dapat di-deploy langsung ke Vercel, Netlify, Cloudflare Pages, atau GitHub Pages.
